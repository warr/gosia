\section{XSTATIC}
\label{sect:xstatic}

\noindent This function calculates the static part of the
deorientation.\\

\noindent Here we use:\\

\begin{equation}
h = {1 \over {1 + (0.012008 \beta Z^{0.45})^{5/3}}}
\end{equation}

\noindent and\\

\begin{equation}
\mathrm{QCEN} = Z h^6
\end{equation}

\noindent and\\

\begin{equation}
\mathrm{DQ} = \sqrt{(\mathrm{QCEN} (1 - h))}
\end{equation}

\noindent Then we integrate over the range QCEN $\pm$ 3 DQ to
calculate XNOR.\\
